# Static Framework
Simple static html for use locally

## Installation
Install dependencies
```
npm install
```

### RVM
You can use a separate gemset using RVM. There is a `.rubyversion` file that currently has the following version listed `2.1.1@static-framework`

To activate this version:
```
rvm use 2.1.1@static-framework
```

## Run program
On the root directory type the following:
`gulp`
